Device Pairing

Generate a pairing code and enter it in the RuneLite plugin to link a player's client to the group. Each code expires after 5 minutes.

Setup

This app requires each group member to install a RuneLite plugin from the Plugin Hub in order to track player information. After installing the plugin, use device pairing to link each player's client to the group.

Steps:

  1. Install the RuneLite plugin from the Plugin Hub.
  2. Click "Generate Pairing Code" above to get a 5-digit code.
  3. Enter the code in the RuneLite plugin to pair your device.
  4. Once paired, player data is automatically ingested.

Use the provided group credentials to log in on the website and manage your group.
